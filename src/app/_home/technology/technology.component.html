Technology

This is a summary of the technology used by this website.

The Technology Stack

Client

  • Angular 131 and TypeScript single-page application
  • Bootstrap and Angular Material components libraries.

Server

  • ASP.NET 62 and C# REST API
  • Azure 'serverless' SQL Database

Deployment

  • Hosted on the Azure cloud platform in Docker containers
