About me

.NET/C#

  • Websites with ASP.NET 6. I build REST APIs with the ASP.NET and MVC libraries.

  • Entity Framework ('code-first'), LINQ, and AutoMapper for object-database mapping.

Web/Client

Databases

  • Designing, developing and deploying SQL databases.

  • Programming stored procedures, queries, and data transformations with T-SQL scripts.
